# Job Importer — Automated Job Feed Processing System

##  Overview

This project automatically imports job listings from external job feeds, processes them in the background, and stores them in a database.  
It includes:
1-Frontend-Nextjs
2-Backend-Nodejs, Express.js
3-DB-MongoDB
4-Other service-Redis, BullMq, Worker, SWR
---

## ⚙️ Features

1- import the job from external feed url
2- use redis and BullMq fro background processing
3- store all jobs and import logs in mongodb
4- Build the admin dashboard using Next.js
5- view the history of import data
